制作期間：2025年8月 ～ 2026年1月
プレイ人数：1人 ～ 4人
使用言語：C++ / HLSL

ゲーム内容

『CrazyGuys』は、他のプレイヤーよりも早くゴールに到達することを目的とした、 最大4人対戦の3Dパーティーアクションゲームです。

  • プレイヤーを弾き飛ばすギミック
  • タイミングを見極めて進むトラップ

操作精度と状況判断が求められるゲーム性となっています。

プレイ動画（PV）

工夫点・アピールポイント

責務分離を意識したクラス設計

プレイヤーの「状態・挙動管理」と「入力処理」を分離して設計しています。

  • Player：状態管理・アニメーション・当たり判定
  • PlayerControl：プレイヤー入力処理
  • AIControl：AI操作処理

Jsonによるデータドリブン設計

ギミックやステージ配置はすべて Json ファイルで管理し、 コード修正なしで調整できる構成にしています。

FadeManager / SoundManager の集中管理

フェード演出・サウンド再生はそれぞれ Manager クラスで一元管理し、 シーン遷移時の不具合や処理の分散を防いでいます。

今後の展望

オフスクリーンレンダリングによる画面分割

各プレイヤー視点をレンダーターゲットに描画し、 UV分割によって柔軟な画面分割マルチプレイを実現する構想です。
